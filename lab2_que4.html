Introduction

This web page is designed with an external CSS file to give it a clean and consistent style. Using a universal selector in the CSS, we apply some basic styles to all elements on the page. This makes it easy to maintain a consistent look across the entire site.

Why Use External CSS?

External CSS allows us to separate content from design. It makes our HTML code cleaner and easier to read. The same stylesheet can be reused for multiple pages, saving time and effort. With the help of a universal selector, we can apply base styles to all elements effortlessly.
